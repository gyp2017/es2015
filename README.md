# ECMAScript2015

개인적으로 자바스크립트를 공부하며 적는 기록임.

개발 환경 (2017.5.20)
- macOS Sierra (v10.12.5)
- Google Chrome (v58)
- Visual Studio Code (v1.12.2)
- Node.js (v6.10.2)

참고 링크
- [ECMAScript 6](http://www.aladin.co.kr/shop/wproduct.aspx?ItemId=100942112)
- [Babel](https://babeljs.io/)

서버 실행
```sh
$ npm start
```
or
```sh
$ yarn start
```
